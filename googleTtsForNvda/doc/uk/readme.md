# Google TTS для NVDA

Додаток для NVDA, що використовує синтез мовлення WebAssembly (WASM) із Google Chrome для отримання якісних природних голосів в офлайн-режимі.

## Особливості

* **Комплексна підтримка голосового керування**: Підтримує всі мови та голоси, доступні у WasmTtsEngine. Це включає пакети Chrome OS (оптимізовані для частого використання та високошвидкісного читання) та пакети Google Natural (розроблені для вищої якості, стандартного читання тексту).
**100% офлайн-мовлення**: мовлення генерується локально за допомогою фонового процесу Google Chrome.
* **Низька затримка**: Використовує вдосконалену сегментацію тексту (менше перше речення) для забезпечення миттєвих голосових відповідей.
* **Тимчасовий аудіокеш**: кеш у пам’яті для коротких фраз (до 200 символів), що дозволяє безпечно оптимізувати повторювані фрази.
* **Графічний інтерфейс менеджера голосів**: Перевірка, завантаження або видалення голосів пакетами за допомогою інтерфейсу з множинним вибором прапорців.
* **Фонові операції**: Неблокуюче завантаження та видалення у фонових потоках.
* **Швидкий доступ**: Натисніть **`NVDA+Ctrl+Shift+G`**, щоб миттєво відкрити «Менеджер голосів».

## Встановлення та перший запуск

1. Завантажте найновіший пакет `.nvda-addon` зі сторінки [Випусків](https://github.com/nguyenanhduc09/Google-TTS-For-NVDA/releases).
2. Відкрийте пакет (або скористайтеся меню NVDA «Магазин додатків» -> «Встановити з зовнішнього джерела») і дотримуйтесь вказівок для його встановлення.
3. Під час першого вибору **Google TTS для NVDA** як синтезатора, якщо голоси не встановлено, NVDA повідомить вас про це та автоматично відкриє діалог **«Менеджер голосів Google TTS...»**, щоб ви могли завантажити принаймні один голос для використання.
4. Крім того, ви також можете натиснути **`NVDA+Ctrl+Shift+G`** або перейти до **Меню NVDA -> Інструменти -> Менеджер голосів Google TTS...** у будь-який час, щоб керувати своїми пакетами.
5. Позначте прапорці біля потрібних голосів і натисніть **Завантажити вибрані голоси**.

## Налаштування конфігурації

Синтезатор підтримує стандартне кільце налаштувань мовлення NVDA:
* **Голос**: Виберіть один із завантажених пакетів голосів/мов.
* **Швидкість**: Швидкість мовлення (відповідає діапазону швидкості в Chrome від 0,35x до 2,0x).
* **Додаткове прискорення**: Увімкніть цю функцію, щоб подвоїти розраховану швидкість мовлення для швидкого читання.
* **Висота**: Регулювання тону мовлення.
* **Гучність**: Гучність мовлення (відповідає діапазону гучності в Chrome від 0,0 до 1,0).
